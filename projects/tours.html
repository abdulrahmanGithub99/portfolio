موقع حجز رحلات سياحية

قمت بتطوير موقع لحجز الرحلات السياحية متكامل يتيح للافراد عرض الرحلات والقيام بحجزها عبر الإنترنت بطريقة سهلة وآمنة.
يتضمن الموقع لوحة إدارة حجوزات متكاملة،لوحة للاحصائيات، نظام ادارة الرحلات ، وإدارة العملاء، تحكم بصلاحيات لوحة التحكم، اعدادات الموقع مع التحكم بعرض الشرائح.
استخدمت في هذا المشروع Html, Css, JS للواجهة الأمامية PHP Laravel للواجهة الخلفية MySql لقاعدة البيانات.

المميزات

  • واجهة مستخدم سهلة الاستخدام
  • نظام إدارة للرحلات متكامل
  • لوحة تحكم للإداريين
  • تحكم كامل بصلاحيات لوحة التحكم
  • توافق مع الأجهزة المحمولة
  • احصائيات مفصلة
  • أمان وحماية البيانات
  • تجربة مستخدم سلسة
  • نظام إشعارات للمستخدمين وادارة الموقع
