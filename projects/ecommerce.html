متجر الكتروني

قمت بتطوير متجر إلكتروني متكامل يتيح للشركات عرض وبيع منتجاتها عبر الإنترنت بطريقة سهلة وآمنة.
يتضمن المتجر نظام إدارة منتجات متكامل، وسلة تسوق، ونظام دفع آمن، وإدارة للمخزون، شركات شحن، بالإضافة إلى لوحة تحكم للإداريين.
استخدمت في هذا المشروع Html, Css, JS للواجهة الأمامية PHP Laravel للواجهة الخلفية MySql لقاعدة البيانات، مع دمج Paypal لمعالجة المدفوعات.

  • واجهة مستخدم سهلة الاستخدام
  • نظام إدارة منتجات متكامل
  • نظام دفع آمن
  • إدارة المخزون
  • شركات شحن
  • لوحة تحكم للإداريين
  • توافق مع الأجهزة المحمولة
  • تحليلات وتقارير مفصلة
  • دعم فني مستمر

  • تجربة مستخدم سلسة
  • توافق مع جميع المتصفحات
  • إمكانية تخصيص التصميم
  • نظام تقييم ومراجعة المنتجات
  • نظام إشعارات للمستخدمين
