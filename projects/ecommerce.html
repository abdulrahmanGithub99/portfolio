متجر الكتروني

مشروع متجر إلكتروني مبني باستخدام Laravel 10 وLivewire 3، يهدف إلى تقديم منصة متكاملة لإدارة وبيع المنتجات إلكترونيًا مع لوحة تحكم إدارية كاملة.
استخدمت في هذا المشروع Html, Css, JS للواجهة الأمامية PHP Laravel للواجهة الخلفية MySql لقاعدة البيانات.

المميزات

  • واجهة مستخدم سهلة الاستخدام
  • لوحة تحكم للإداريين
  • توافق مع الأجهزة المحمولة
  • أمان وحماية البيانات
  • نظام إشعارات للمستخدمين وادارة الموقع
  • فلاتر بحث ديناميكية بالاعتماد على البحث الفوري والمطابقات الذكية
  • إدارة الوسائط: رفع الصور وتعديلها
  • لوحة تحكم تفاعلية لضبط البيانات وعرض الإحصاءات
  • نظام الإشعارات الفورية لعرض جديد الطلبات
  • إدارة الأدوار والصلاحيات للتحكم الدقيق في الوصول
  • تقارير ورسوم بيانية بسيطة لعرض إحصاءات المبيعات والطلبات
  • نظام تحقق وإدارة جلسات المستخدمين مع تسجيل الدخول وتغيير كلمة المرور واستعادة الحساب
